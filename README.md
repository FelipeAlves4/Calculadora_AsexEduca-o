# Calculadora de Resultados ASEX Educação

Aplicação web responsiva para comparar o cenário financeiro atual de restaurantes e negócios de alimentação com um cenário projetado após melhorias operacionais.

## Como executar

1. Instale as dependências:

```bash
npm install
```

2. Rode o servidor de desenvolvimento:

```bash
npm run dev
```

3. Gere uma versão de produção:

```bash
npm run build
```

## Recursos

- Dois cenários financeiros sincronizados: atual e projetado.
- Campos em reais, percentuais e sliders com atualização imediata.
- Cadastro detalhado de despesas fixas e operacionais.
- Cards comparativos, tabela de variações e gráficos em tempo real.
- Preenchimento com exemplo, cópia do cenário atual, simulação de melhoria, impressão/PDF e modo apresentação.
- Salvamento temporário no `localStorage`.
